# Descripción del proyecto
"La receta del opa" será una pagina web creada con HTML junto con CSS y Javascript, con este foro, buscamos entregar a la gente un canal por el cual compartir y recrear resetas caseras, para distintas ocasiones.

# Instalación

1. Clona este repositorio en tu computador:
   ```sh
   git clone https://github.com/tu_usuario/La_receta_del_opa.git
   ```
2. Abre la carpeta del proyecto en tu editor de preferencia (recomendado: Visual Studio Code).
3. No se requiere instalación de dependencias, ya que el proyecto es HTML, CSS y JavaScript puro.
4. Para probar la funcionalidad de Supabase, debes configurar tus credenciales en los archivos correspondientes de JavaScript (ver documentación interna).
5. Abre el archivo `La_Receta_Del_Opa/index.html` en tu navegador para comenzar a usar la aplicación.

# Despliegue

Para ejecutar este proyecto en otro dispositivo:

1. Descarga o clona el repositorio.
2. Abre la carpeta `La_Receta_Del_Opa` y ejecuta el archivo `index.html` en tu navegador.
3. Si deseas probar la conexión con la base de datos (Supabase), asegúrate de tener configuradas las variables y credenciales necesarias en los archivos JavaScript.
4. No se requiere servidor backend para la visualización básica, pero para funcionalidades completas (registro, login, publicación) es necesario tener acceso a la base de datos configurada.
